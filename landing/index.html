🚛 ทะเบียนรถ (Vehicle Management)

ลงทะเบียนรถทุกประเภท ข้อมูลครบถ้วน

  • รองรับ: 4ล้อ, 6ล้อ, 10ล้อ, หัวลาก, กระบะ
  • ประกันภัย / ภาษีรถ / พ.ร.บ. — แจ้งเตือนก่อนหมดอายุ
  • ประวัติซ่อมบำรุง ทุกครั้ง
  • สุขภาพรถ: เขียว / เหลือง / แดง
  • เอกสาร: ทะเบียน, กรมธรรม์ (เก็บใน R2)
  • รองรับรถเช่า + รถร่วม

👤 คนขับ (Driver Management)

จัดการคนขับ ตารางเวร KPI คะแนน

  • ข้อมูลครบ: ใบขับขี่, บัตร กรมขนส่ง, สุขภาพ
  • KPI Score 0-100 (ตรงเวลา, ประหยัดน้ำมัน, Rating)
  • ตารางเวร + วันลา + OT
  • คำนวณเงินเดือนอัตโนมัติ (ฐาน + เบี้ยเลี้ยง + โบนัส)
  • ประวัติอุบัติเหตุ + ฝ่าฝืน
  • AI แนะนำคนขับที่เหมาะสม

📋 เที่ยววิ่ง (Trip Management)

จัดเที่ยว มอบหมาย ติดตาม คำนวณต้นทุน/กำไร

  • สร้างเที่ยว → มอบหมายรถ+คนขับ → ติดตาม
  • Status flow: draft → pending → started → completed
  • หลายจุดส่งในเที่ยวเดียว
  • คำนวณค่าขนส่งอัตโนมัติ (Longdo Map)
  • ต้นทุน: น้ำมัน + ทางด่วน + คนขับ = กำไร/ขาดทุน
  • POD: รูปถ่าย + ลายเซ็นดิจิทัลผู้รับ

📍 GPS Tracking Real-time

ติดตามรถทุกคันแบบ real-time บนแผนที่

  • GPS จาก Driver App ทุก 30 วินาที
  • แผนที่ Longdo Map แสดงรถทุกคัน
  • ดูเส้นทางย้อนหลัง (Trail)
  • ความเร็ว, ทิศทาง, แบตเตอรี่
  • แจ้งเตือน Geofence + ขับเร็วเกิน
  • WebSocket สำหรับ real-time update

🔧 ซ่อมบำรุง (Maintenance)

ใบสั่งซ่อม อนุมัติ ติดตาม สต๊อกอะไหล่

  • 3 ประเภท: ซ่อมตามรอบ / ตามอาการ / ฉุกเฉิน
  • Approval flow: draft → รออนุมัติ → อนุมัติ → เสร็จ
  • คิดค่าอะไหล่ + ค่าแรง รายชิ้น
  • รูปก่อน/หลังซ่อม
  • สต๊อกอะไหล่ + แจ้งเตือนเมื่อต่ำกว่า min
  • เชื่อม BC Account บัญชี (journal entry)

🤝 รถร่วม (Partner Vehicles)

ลงทะเบียนรถร่วม จอง จ่ายเงิน AI จับคู่

  • ลงทะเบียนรถร่วม: ข้อมูลเจ้าของ + รถ + คนขับ
  • ราคา: ต่อเที่ยว / ต่อ กม. / ต่อวัน
  • ค้นหารถว่าง ตาม zone + ประเภท + วัน
  • AI Matching Score 100 คะแนน
  • จ่ายเงิน + หัก ณ ที่จ่าย (ภ.ง.ด.3)
  • Rating + ประวัติเที่ยว

💰 ต้นทุนขนส่ง (Cost Analysis)

P&L ต่อคัน/เดือน รายงานน้ำมัน วิเคราะห์กำไร

  • P&L ต่อคัน: รายได้ vs ต้นทุน (น้ำมัน, ทางด่วน, ซ่อม, คนขับ)
  • รายงานน้ำมัน: ลิตร, ราคา, กม./ลิตร ต่อคัน
  • ต้นทุนต่อเที่ยว + กำไรต่อเที่ยว
  • อัตราการใช้รถ (Vehicle Utilization)
  • เปรียบเทียบรายเดือน/รายสัปดาห์
  • Export Excel / PDF

⚠️ แจ้งเตือนอัตโนมัติ (Alerts)

Cron job ตรวจทุกชั่วโมง แจ้งเตือน 7 ประเภท

  • ประกันภัยใกล้หมดอายุ (30/15/7 วัน)
  • ภาษีรถใกล้กำหนด
  • พ.ร.บ. ใกล้กำหนด
  • ใบขับขี่หมดอายุ
  • ซ่อมบำรุงครบกำหนด (กม. / วัน)
  • ขับเร็วเกินกำหนด + ออกนอก Geofence

🤖 AI Agent + LINE Bot

ลูกค้าทัก LINE → AI ตอบทันที จองรถ ติดตามงาน

  • Claude AI + 46 MCP Tools
  • ค้นหารถว่าง คำนวณราคา จองเที่ยว อัตโนมัติ
  • ตอบภาษาไทย เข้าใจบริบทขนส่ง
  • Rich Menu: ดูรถว่าง / จอง / ติดตาม / แจ้งซ่อม / ต้นทุน / AI
  • Push Notification: งานใหม่, ส่งเสร็จ, อนุมัติ
  • Reply ฟรีไม่จำกัด (LINE Messaging API)

🔗 UCP + A2A Gateway

Universal Commerce Protocol — AI จากระบบอื่นจองรถได้

  • Service Catalog: ประเภทรถ + ราคา
  • Availability Check: รถว่าง ตาม zone + วัน
  • Booking + Payment (PromptPay / Stripe)
  • Fulfillment: ติดตาม real-time + POD
  • A2A Agent Card: AI-to-AI communication
  • รองรับ Claude Desktop, OpenClaw, HiClaw

🗺️ Longdo Map Integration

แผนที่ไทยครบ — Geocoding, Routing, Search

  • Geocode: ที่อยู่ → พิกัด
  • Reverse Geocode: พิกัด → ที่อยู่
  • Route: คำนวณเส้นทาง + ระยะทาง + เวลา
  • Search: ค้นหาสถานที่ใกล้เคียง
  • แผนที่ไทยถูกกว่า Google Maps
  • Turn-by-turn Navigation (Driver App)

📊 Dashboard + KPI + Reports

สรุปภาพรวม ตัวเลขสำคัญ รายงานครบ

  • รถทั้งหมด / active / maintenance / critical
  • เที่ยววิ่งวันนี้ / รายได้ / ต้นทุน / กำไร
  • Vehicle Utilization Rate
  • On-time Delivery Rate
  • Fuel Efficiency (กม./ลิตร)
  • Driver Leaderboard (อันดับคนขับ)
